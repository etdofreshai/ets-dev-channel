# IDEAS

## Product Name
- **Paiload** (paiload.com) — Personal AI + Payload

## Free vs Paid Tiers
- Free: 3 channels, 10MB attachment limit, local storage only
- Paid: unlimited channels, 100MB attachments, cloud sync, API access
